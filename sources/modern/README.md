# Modern Scholarship — How to Obtain

These works are copyrighted and cannot be stored in the repository.
Below are instructions and direct links for accessing each referenced modern work.

**For citation verification:** Borrow from Internet Archive or access the linked sources,
then check the cited passage manually.

## Access Legend

- **IA Borrow** = Internet Archive 1-hour or 14-day digital borrow (free account required)
- **Open Access** = Free PDF download
- **Preview** = Google Books partial preview
- **Paywalled** = Requires institutional access or purchase
